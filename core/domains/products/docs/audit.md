# Audit Requirements

## Audited Actions
- Product creation
- Product updates
- Publishing / unpublishing
- Deletion
- Pricing changes

## Audit Fields
- Actor (user/system)
- Action
- Timestamp
- Previous state
- New state

## Storage
Audit logs must be immutable and retained per compliance policy.

## Purpose
- Regulatory compliance
- Forensic investigation
- Dispute resolution
